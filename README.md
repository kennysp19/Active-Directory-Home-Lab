# Active Directory Home Lab

## Overview
I built a hands-on Active Directory lab to simulate a real-world enterprise environment and develop foundational skills in identity and access management, user administration, and security monitoring relevant to a SOC Analyst role.


## Objective
To understand how user accounts, permissions, and authentication work in a domain environment, and how these activities generate logs that can be monitored for security events.


## Lab Environment

**Virtualization Platform:**
- VirtualBox

**Systems Configured:**
- Windows Server 2022 (Domain Controller)
- Windows 10 (Client Machine)
